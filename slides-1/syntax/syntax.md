## default parameters ##
```scala
def hello(name:String, end: String = "!") =
  "Hello " + name + end
	
hello("world") 
// Hello world!
	
hello("world", ", whats up?") 
// Hello world, whats up?
```

## symboler ##
```scala
val a = BigDecimal("100.00")
val b = BigDecimal("25.00")
	
a * b  // 2500.0000
a - b  // 75.00
a / b  // 4
a + b  // 125.00
a > b  // true
a < b  // false
a >= b // true
a <= b // false
```
